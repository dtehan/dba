  <action>
Create `src/main/services/mcp-schema.ts`:

1. Import `getMcpUrl` from `./mcp-manager`.

2. Module-level `let mcpSessionId: string | null = null;` to track MCP session state.

3. Create `async function initializeMcpSession(): Promise<void>`:
   - POST to `getMcpUrl()` with JSON-RPC `initialize` (same pattern as mcp-manager.ts testTeradataConnection):
     ```json
     { "jsonrpc": "2.0", "id": 1, "method": "initialize", "params": { "protocolVersion": "2024-11-05", "capabilities": {}, "clientInfo": { "name": "teradata-dba-agent", "version": "1.0.0" } } }
     ```
   - Headers: `{ 'Content-Type': 'application/json', 'Accept': 'application/json, text/event-stream' }`
   - On success: check response headers for `Mcp-Session-Id` ŌĆö if present, store in `mcpSessionId`
   - On failure: set `mcpSessionId = null` (server may be stateless, proceed anyway)

4. Create `async function callMcpTool(toolName: string, args: Record<string, string> = {}): Promise<string>`:
   - If `mcpSessionId === null`, call `initializeMcpSession()` first
   - POST to `getMcpUrl()` with:
     ```json
     { "jsonrpc": "2.0", "id": Date.now(), "method": "tools/call", "params": { "name": toolName, "arguments": args } }
     ```
   - Headers: `{ 'Content-Type': 'application/json', 'Accept': 'application/json, text/event-stream' }` plus `'Mcp-Session-Id': mcpSessionId` if mcpSessionId is set
   - If response is 404: reset `mcpSessionId = null`, call `initializeMcpSession()`, retry once
   - Parse response JSON: `{ result?: { content: Array<{ type: 'text'; text: string }> }; error?: { message: string } }`
   - If error: throw `new Error(json.error.message)`
   - Return `json.result?.content.map(c => c.text).join('\n') ?? ''`
   - Wrap in try/catch, timeout after 15 seconds with AbortController

5. Create `export async function listDatabases(): Promise<string[]>`:
   - Call `callMcpTool('base_databaseList')`
   - Parse result: split by newline, filter empty strings, trim each line
   - Return array of database names

6. Create `export async function fetchSchemaContext(databaseName: string): Promise<string>`:
   - Call `callMcpTool('base_tableList', { databaseName })` to get table names
   - Parse table names from result (split by newline, filter empty, trim)
   - For each table (up to 20 tables to stay within token budget):
     - Call `callMcpTool('base_columnDescription', { databaseName, tableName })`
   - Use `Promise.allSettled` for column fetches (don't fail all if one table errors)
   - Build formatted context block:
     ```
     Database: {databaseName}
     Tables: {tableCount}

     ### {tableName1}
     | Column | Type |
     |--------|------|
     | col1 | VARCHAR(100) |
     | col2 | INTEGER |

     ### {tableName2}
     ...
     ```
   - Cap total output at approximately 4000 characters (~4K tokens). If exceeding, truncate table list and append `... and {N} more tables (select a specific table for details)`
   - Return the formatted string

7. Export `listDatabases` and `fetchSchemaContext`.
